To update the titles, input some text and press ENTER KEY (its triggered on change):

If you want it to change on keyup or other event, follow the simple example.

You can open the console and execute:
user.name.value = "some text"; And you will see the input and the 2 titles change. Additionally, try changing the "value" attribute in HTML directly, and it will also change the titles.

Explanation:

const user = m2d2(".user", { name: { value: "Hello" } }); const title = m2d2("h1", { text : [ user.name , "value" ] });

The important point is that we need to assign an array made of a m2d2 object and the property you want to assign. If you set: text : user.name.value it will set the value only, and it won't be updated if it changes.
