1. How to start

In this example we will learn how to start using M2D2 with a simple user profile. We "fill" the fields using a javascript plain object.

Lessons:

  1. We start by using the function: m2d2.ready() which executes your instructions when the page (DOM) is ready.
  2. The dollar sign in the callback function: m2d2.ready($ => { $("#profile"); }) is the suggested symbol for M2D2 (and the one used across all examples and documentation). You can use anything you want, for example: m2d2.ready(m => { m("#profile"); })
  3. Usually you want to use an ID as selector: $("#profile") but you can use any supported selector, for example: $("body section div:first-child") Although I don't recommend it because your code will perform better and it will be easier to understand if you stick to IDs (more about this later in this tutorial).
  4. You don't need to specify all parents to find an element, for example, instead of: $("#profile", { details : { age : 80 }) can be set in a more direct way: $("#profile", { age : 80 });
