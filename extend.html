Extending M2D2 Demo

This text will be replaced in 3 seconds... wait for it.

This text was shown using an extension property 'show', and added some style.

Source code:

body = m2d2({
	original : {
		color : "red",
		show : true
	},
	replace : {
		css : "myclass",
		bgcolor: "#EEE",
		color: "green",
		show : false
	}
});
setTimeout(function(){
	body.original.show = false;
	body.replace.show = true;
}, 3000);
